# Frontend Mentor - Bookmark Landing Page Solution

This is a solution to the [Bookmark landing page challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/bookmark-landing-page-5d0b588a9edda32581d29158). Frontend Mentor challenges help you improve your coding skills by building realistic projects.

A responsive Frontend Mentor challenge built with **React**, **Vite**, and **Tailwind CSS**.

## Tech Stack

- React (hooks)
- Vite
- Tailwind CSS
- JavaScript (ES6+)

## Features

- Feature tabs with dynamic image/text switching
- Email validation with error state display
- FAQ accordion with smooth open/close animation
- Mobile burger menu with full-screen overlay
- Download cards with staggered layout
- Fully responsive layout (desktop/tablet/mobile)

## Run Locally

```bash
git clone https://github.com/p3lm3shka1/bookmark-landing-page.git
cd bookmark-landing-page
npm install
npm run dev
```
